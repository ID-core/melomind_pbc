## About Melomind

Melomind is a collaborative study and work space designed to help students and teams stay connected, focused, and productive.  
It combines real-time communication, shared tools, and a customizable interface to make online studying or working feel engaging rather than isolating.

---

## Key Features

- Real-time chat with complete visibility of active room members  
- Collaborative notes with automatic real-time syncing to Google Drive  
- Instant switching to video or voice calls, including screen sharing  
- Admin panel to manage rooms and participants  
- Shared music controls, including track selection and volume adjustment  
- Light and dark mode support with streak tracking  
- Ability to create or join public and private rooms (with password protection)

---

## Project Notes

A significant part of this project involved learning WebRTC and Socket.IO from scratch.  
These technologies now power the real-time communication and collaboration features at the core of Melomind.
